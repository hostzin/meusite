## Guia de Cursos para iniciar e se especializar na carreira de T.I

Mini-site estático desenvolvido como atividade da disciplina **Internet e Aplicações Web – Progressão Parcial** do curso de **Técnico em Desenvolvimento de Sistemas (TDS)** do IFPI.

O objetivo do site é apresentar alguns cursos locais e presenciais da área de Tecnologia da Informação, divididos em três categorias:

- **Suporte técnico**
- **Infraestrutura (Redes)**
- **Analista de sistemas**

Cada seção traz uma breve explicação da área, cursos recomendados, imagens ilustrativas e, ao final, uma **tabela com a média salarial** de cada carreira, além de links para páginas oficiais (SENAC, SENAI e IFPI).

## Tecnologias utilizadas

- HTML
- CSS

## Estrutura do projeto

- `index.html` → página principal do mini-site  
- `estilos.css` → arquivo de estilos (cores, fontes, layout)  
- pasta `Imagens/` → imagens utilizadas no site  

## Como visualizar o projeto

1. Abra o arquivo `index.html` em qualquer navegador **ou**
2. Acesse a versão publicada no **GitHub Pages**:
